# SIBF 2026 Map

2026 서울국제도서전 부스배치도 웹 앱입니다.

출판사명이나 부스번호로 참가사를 검색하고, 지도에서 위치를 확인하며, 방문하고 싶은 부스를 저장할 수 있습니다.

## Link

https://sibf2026map.pages.dev

## Features

- A홀/B1홀 부스 지도 탐색
- 출판사명, 영문명, 부스번호 검색
- 부스 선택 시 지도에서 위치 확인
- 관심 부스 저장
- 부스별 메모 및 사진 기록
- 모바일 화면 대응

## Mobile Usage

모바일 브라우저에서 접속한 뒤 공유 메뉴를 열고 `홈 화면에 추가`를 선택하면 앱처럼 사용할 수 있습니다.
